# nvim

## Installation

Install Telescope dependencies:

- ripgrep
- fd-finder

For Wayland, install:

- wl-clipboard

Run :PackerSync.

Run :Mason and be sure are installed:

- clangd 
- css-lsp cssls
- eslint-lsp eslint
- html-lsp html
- intelephense 
- jdtls 
- json-lsp jsonls
- lua-language-server lua_ls
- netcoredbg 
- omnisharp 
- prettier 
- pyright 
- rust-analyzer rust_analyzer
- tailwindcss-language-server tailwindcss
- typescript-language-server tsserver
- stylua
